What Do We Use Trig For?

Often in school you may wonder why you are learning a certain subject. This page will be devoted to explaining what trigonometry is used for in the real world and why you should learn about it. In the real world you may not be suddenly asked what sin(π/4) is equal to, but if you have interest in science or mathematics then you will find a plethora of uses with trigonometry.

Games

It may seem odd, but trigonometry is very useful for developing video games. One of these uses can be seen in the demo on the first page of the trigonometry section of this website. Often it is useful to have a line point towards an object on the screen, and that can be achieved using trigonometry. On the demo on the first page, the position of the mouse is stored as a set of x and y coordinates. I wanted to generate a line from the center of the screen in the direction of the mouse and reuse this angle when shooting projectiles. This was achieved by using the inverse tangent function with those coordinates as inputs. Furthermore, it is useful to know how far away the mouse is from the center of the screen and that can also be solved using trigonometry. The distance was calculated using the pythagorean theorem (a^2 + b^2 = c^2) which is a trigonometric identity. Using sine we could develop platforms in a platformer that smoothly moves up and down. Trigonometry is often used in figuring out where to look and fire based off of mouse input. Essentially, any time we need to know the relationship between two positions, and in particular the angle between them, we will need trigonometry. User Interfaces

When developing a user interface for a program or website, sometimes you want an element to appear on screen. Making it suddenly show on the screen can be jarring, so you may want to have it smoothly glide onto the screen. This could be done with interpolation, which smoothly transitions something from some condition A to some other condition B. There are many ways to interpolate, but a very appealing way is to use a sine wave. Sine waves begin at 0 when you input 0, and reach a value of 1 when you input π/2 (or 90 degrees). Using this number from 0-1 as a percentage, you could make something slide smoothly across the screen.

Physics

In physics there are countless ways trigonometry is used. Imagine placing a box on a large flat platform. Gravity pulls this box towards the center of the earth, but the platform it is on prevents it from moving. Now imagine you lift one end of this platform. At some point, the box will begin sliding down the platform. When the box is perpendicular to the ground, the box simply falls. Clearly, the rate of acceleration of this box changes as you increase the angle between the ground and the platform. This relationship between the angle and the acceleration can be calculated using trigonometry. Trigonometry also can be used when there is simple harmonic motion. Under ideal conditions, the swing of a pendulum or the bounce of a spring can be represented as a smooth repeating motion. This motion can be modelled using trigonometry. The orbit of planets often can be modelled this way, but there are many more uses for trigonometry when modelling the orbits of planets. In fact, trigonometry allows us to calculate the distance from the earth to the sun!
